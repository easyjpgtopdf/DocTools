# Backup Files

This folder contains backup copies of important files for recovery purposes.

## Resume Maker Backup

- **resume-maker-latest-backup.html** - Latest backup of resume-maker.html (always updated)
- **resume-maker-backup-YYYY-MM-DD-HHMMSS.html** - Timestamped backups for version history

## Biodata Maker Backup

- **biodata-maker-latest-backup.html** - Latest backup of biodata-maker.html (always updated)
- **biodata-maker-backup-YYYY-MM-DD-HHMMSS.html** - Timestamped backups for version history

## Recovery Instructions

### If resume-maker.html gets corrupted or deleted:

1. Copy `resume-maker-latest-backup.html` to the root directory
2. Rename it to `resume-maker.html`
3. Or use a timestamped backup if you need a specific version

### If biodata-maker.html gets corrupted or deleted:

1. Copy `biodata-maker-latest-backup.html` to the root directory
2. Rename it to `biodata-maker.html`
3. Or use a timestamped backup if you need a specific version

## Note

These backups are automatically committed to GitHub for additional safety.
