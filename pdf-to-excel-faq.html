General Questions

What is PDF to Excel Converter?
PDF to Excel Converter is an online tool that converts PDF files into Excel spreadsheets (XLSX format). It extracts tables and data from PDF documents, allowing you to edit, analyze, and work with the data in Excel.
Is PDF to Excel conversion free?
Yes! We offer a FREE version that works for text-based PDFs with simple tables. The free version uses browser-based conversion (no cloud costs) and is perfect for basic PDFs.
Free Version: Zero cost, works for text-based PDFs with simple tables. No credits required.
What file formats are supported?
We support PDF files only as input. The output is always in Excel format (XLSX). You can convert:
  • Text-based PDFs (with selectable text)
  • Scanned PDFs (requires Premium OCR)
  • PDFs with tables
  • Multi-page PDFs
What is the maximum file size?
The maximum file size is 100 MB per PDF file. For larger files, consider splitting them into smaller parts.

When should I use Free version?
Use the Free version when:
  • Your PDF is text-based (not scanned)
  • Tables are simple (no merged cells, basic structure)
  • You don't need high accuracy
  • You want zero cost conversion
Tip: Try the free version first. If it doesn't work well, upgrade to Premium for OCR-powered conversion.
When should I use Premium version?
Use the Premium version when:
  • Your PDF is scanned (image-based)
  • Tables are complex (merged cells, multi-line headers)
  • You need high accuracy
  • Converting bank statements, invoices, or financial reports
  • Free version conversion failed or gave poor results
How do I know if my PDF needs Premium?
Our system automatically detects if your PDF needs Premium:

Automatic Detection: The system analyzes your PDF and shows an upgrade popup if Premium OCR is needed.

ID Card Conversion

Can I convert ID cards to Excel?
ID cards cannot be converted to Excel because they don't contain tables. However, we offer special ID card processing:
  • Free Users: Text extraction + CSV template download
  • Premium Users: Structured data extraction using identity-docai processor
Note: ID cards are automatically detected and routed to a special processing endpoint. Excel conversion buttons are disabled for ID cards.
What happens when I upload an ID card?
When an ID card is detected:
  • Excel conversion buttons are automatically disabled
  • ID card-specific buttons appear: "View Text" and "Download CSV Template"
  • Free users get text extraction (zero cost)
  • Premium users get structured data extraction (6 credits/page)
What data can be extracted from ID cards?
ID card extraction includes:
  • Name
  • Date of Birth
  • ID Number
  • Gender
  • Address
  • Father's Name / Mother's Name
  • Nationality
  • Issue Date / Expiry Date
Premium Feature: Structured extraction with high accuracy using Google Document AI identity processor.

Technical Questions

How does the conversion work?
Free Version:
  • Uses PDF.js (browser-based) to extract text
  • Detects table structure from text positioning
  • Creates Excel using SheetJS (client-side)
  • No cloud processing, zero cost
Premium Version:
  • Uploads PDF to Google Cloud Storage
  • Processes with Google Document AI OCR
  • Extracts tables and structured data
  • Creates high-accuracy Excel file
  • Returns download link
Is my data secure?
Yes, your data is secure:
  • Free Version: Processing happens in your browser. No data is sent to servers.
  • Premium Version: Files are temporarily stored in Google Cloud Storage, processed, and then deleted automatically.
  • Download links expire after 1 hour
  • No data is stored permanently
  • All processing uses secure HTTPS connections
What OCR technology is used?
Premium conversions use Google Document AI:
  • Enterprise-grade OCR technology
  • Multi-processor support (form-parser, layout-parser, etc.)
  • Automatic processor selection based on document type
  • High accuracy for scanned documents
Processor Selection: The system automatically chooses the best processor:
  • Scanned PDFs → layout-parser
  • Complex tables → pdf-to-excel processor
  • Forms/invoices → form-parser
  • ID cards → identity-docai
Can I convert multiple pages?
Yes! You can:
  • Convert all pages at once
  • Select specific pages to convert
  • Each page becomes a separate sheet in Excel (or merged based on table structure)
Cost: Premium conversion charges per page. For example, a 10-page document costs 10 × (credit cost per page).
What if conversion fails?
If conversion fails:
  • Free Version: Upgrade popup appears if PDF needs OCR
  • Premium Version: Credits are NOT deducted. You can try again.
  • Error message shows the reason
  • Contact support if issue persists
Important: Credits are only deducted after successful conversion. Failed conversions don't cost anything.

Troubleshooting

Conversion is taking too long. What should I do?
Conversion time depends on:
  • File size (larger files take longer)
  • Number of pages
  • Document complexity
  • OCR processing (Premium takes longer than Free)
Typical times:
  • Free version: 5-30 seconds
  • Premium version: 30-120 seconds (for OCR processing)
If it takes longer than 5 minutes, refresh and try again.
Excel file is empty or missing data. Why?
This can happen if:
  • PDF is scanned (use Premium OCR)
  • PDF has no tables (only text content)
  • Tables are too complex for free version
  • PDF is image-based with no text layer
Solution: Try Premium version for scanned PDFs or complex tables. Premium uses OCR to extract data from images.
I'm getting "Insufficient credits" error. What does it mean?
This means:
  • You have less than 30 credits (minimum required)
  • OR you don't have enough credits for the specific document
Example: Converting a 10-page bank statement requires 10 × 2.5 = 25 credits. If you have only 20 credits, you'll see this error.
Solution: Purchase more credits to meet the requirement. Check the pricing table above for credit costs.
Download link expired. What should I do?
Download links are valid for 1 hour for security. If expired:
  • Re-upload the PDF
  • Convert again (credits will be deducted again)
  • Download immediately after conversion
Tip: Download the Excel file immediately after conversion to avoid expiration.
Browser shows "Failed to load resource" error. What's wrong?
This could be due to:
  • Network connectivity issues
  • Backend service temporarily unavailable
  • Browser blocking the request
  • Ad blocker interfering
Solutions:
  • Refresh the page and try again
  • Check your internet connection
  • Disable ad blockers temporarily
  • Try a different browser
  • Wait a few minutes and retry
